# Neural Networks: Zero to Hero

A course on neural networks that starts all the way at the basics.

## Start

```powershell
uv sync --locked
uv run jupyter lab
```

## Route

1. **DL Foundations** — autograd, language modeling, MLPs, initialization, batchnorm, diagnostic, backpropagation, and WaveNet.
2. **GPT stack** — attention, tokenization, GPT-2 reproduction, and the complete LLM pipeline and a one-file microgpt capstone.

See the complete [`book/`](book/) index.
