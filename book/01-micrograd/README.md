# 01 — micrograd

Build backpropagation from scalar operations before moving to tensor-based neural networks.

## Build

A scalar-valued reverse-mode autodiff engine and a small multilayer perceptron.

4. Read [`engine.py`](resources/micrograd/micrograd/engine.py) and [`nn.py`](resources/micrograd/micrograd/nn.py).
5. Watch CS182 [Lecture 1: Introduction](https://www.youtube.com/watch?v=rSY1pVGdZ4I&list=PL_iWQOsE6TfVmKkQHucjPAoRtIJYt8a5A).
